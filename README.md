📩 SMS Spam Classifier

A simple Machine Learning project that classifies SMS messages as Spam or Ham (Not Spam) using NLP techniques.

🚀 Overview

This project focuses on building a machine learning model that can automatically detect whether an SMS message is spam or not based on its content. It uses Natural Language Processing (NLP) techniques to clean and transform text data into numerical features, which are then fed into a Naive Bayes classifier for prediction. The goal is to create a fast, efficient, and beginner-friendly solution for spam detection that demonstrates the practical application of machine learning in real-world communication systems.

🧠 Tech Stack
Python
Pandas
Scikit-learn
TF-IDF Vectorizer
Naive Bayes


⚙️ Workflow
Data Cleaning
Text Preprocessing
Feature Extraction (TF-IDF)
Model Training (Naive Bayes)
Evaluation


📊 Result
Good accuracy for spam detection
Fast and lightweight model
Beginner-friendly project
